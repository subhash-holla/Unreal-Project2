# Unreal-Project2
An open world head-to-head tank combat with simple AI, terrain, and advanced control systems in Unreal 4 Game Engine.

This game is based on [World of Tanks](https://worldoftanks.com/). The build follows the [Udemy course](https://www.udemy.com/unrealcourse/learn/v4/content) on [Unreal 4](https://www.unrealengine.com/en-US/what-is-unreal-engine-4).
